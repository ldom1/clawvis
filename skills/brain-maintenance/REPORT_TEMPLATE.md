# Template du message — Brain Maintenance (Telegram)

Quand le skill **brain-maintenance** envoie un message à Ldom (Telegram), c’est uniquement si une condition l’exige (trim : budget L1 dépassé ; recalibrate : drift détecté). **Il est primordial d’utiliser ce template** pour chaque envoi.

---

## Trim (L1 dépassé)

- **Titre** — ex. « Brain Maintenance — Trim »
- **Résumé** — 1–2 lignes : budget L1 dépassé, fichier(s) ou zone(s) concerné(s), action recommandée (ex. trim manuel, priorisation).

## Recalibrate (drift détecté)

- **Titre** — ex. « Brain Maintenance — Recalibrate »
- **Résumé** — 1–2 lignes : type de drift (SOUL / AGENTS), constat bref, action recommandée.

Rester concis ; ne rien envoyer si tout est nominal.
