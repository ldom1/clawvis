# Template du rapport — Proactive Innovation (Telegram)

Quand le skill **proactive-innovation** envoie un message à Ldom (Telegram), seul ou combiné avec self-improvement dans le cron 4h, le message doit suivre ce format. **Il est primordial d’utiliser ce template** pour chaque envoi. **Un seul** message par run.

---

## Structure (2–4 lignes max)

1. **Learnings** (si combiné avec self-improvement) — 1 ligne : statut learnings (ex. tout vert).
2. **Innovation** — projets scannés ; N améliorations/todos (et tâches Kanban créées) ; M MR créées (avec 1 ligne par MR si M > 0, inclure **lien URL** de chaque MR) ; P idées → entrepreneur.
3. **Lien Kanban** (si pertinent) — URL du Kanban (ex. Hub `/kanban/`).

Exemple : *« 4h — Learnings: tout vert. Innovation: 3 projets scannés, 2 améliorations (hub, debate-arena), 1 MR (hub: nginx try_files) [lien]. 1 idée → entrepreneur. Kanban: [url]. »*
