# Template du rapport — Knowledge Consolidator (Telegram)

Quand le job **Memory Consolidation** (23h30) envoie un résumé à Ldom via Telegram, le message doit suivre ce format. **Il est primordial d’utiliser ce template** pour chaque envoi.

---

## Structure du rapport (à respecter)

1. **Contexte** — Memory Consolidation — [date]
2. **Souvenirs consolidés** — nombre N
3. **Thèmes principaux** — liste courte des thèmes ou domaines de la synthèse du jour
4. **Statut QMD** — bref (ex. index à jour, en cours, erreur)
5. **Insight du jour** (obligatoire si résumé envoyé) — en une phrase : la connaissance ou le fait le plus notable assimilé lors de la collecte/consolidation

Rester concis (quelques lignes).
